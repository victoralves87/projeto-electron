# Projeto Electron js
## Tocador de Musica
## Para rodar em seu computador:

primeiro, clone esse repositório:

```bash
npm install electron electron-builder concurrently wait-on --save-dev
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
